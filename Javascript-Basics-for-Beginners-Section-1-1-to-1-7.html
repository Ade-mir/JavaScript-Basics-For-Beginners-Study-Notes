JavaScript was orginally made for only running in broswers. Every browser has a JavaScript engine that can execute JS code. FireFox has the engine SpiderMonkey, and Chrome has the engine v8.

In 2009 a developer took JS and embedded it in a C++ program and called it Node. therefor allowing it to run anywhere, not just in a browser.

ECMA is an organization that ECMAScript specification that defines the features of JavaScript, and in 2015 they launched a new specifiaction for JS often called ES6.

Seperation of concerns refers to the fact that we want to seperate the JavaScript from the CSS and HTML scripts.
